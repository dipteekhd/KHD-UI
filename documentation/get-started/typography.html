Typography

Use typography to present your design and content as clearly and efficiently as possible.

Font Family

The Nunito and Nunito Sans font will be automatically loaded by KHD-UI for body text and headings respectively.

Font size

Control the font size of an element using the text-{size} utilities.

text-xl

Extra large text

text-lg

Large text

text-md

Medium text

text-sm

Small text

text-xs

Extra small text

Font weight

Control the font weight of an element using the {weight}-font utilities.

light-font

Light font text

regular-font

Regular font text

semibold-font

Semibold font text

bold-font

Bold font text

extrabold-font

Extrabold font text

Spacing - Padding

Control the padding of an element using p-{size}-{direction} utilities.

size : { xs | sm | md | lg | xl | xxl }

direction : { t | b | l | r | vr | hr | all }

p-sm-t
p-sm-b
p-sm-l
p-sm-r
p-sm-vr
p-sm-hr
p-sm-all
