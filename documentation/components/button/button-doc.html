Button

Buttons communicate actions that users can take. They are typically placed throughout your UI, in places like forms,cards,modals etc.

Contained Button

Contained button contain actions that are primary to your app.

Outlined Button

Outlined button contain actions that are important but aren't the primary action in an app.

Text Button

Text buttons are typically used for less-pronounced actions.
