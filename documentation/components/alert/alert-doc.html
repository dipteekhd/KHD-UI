Alert

An alert displays a short, important message in a way that attracts the user's attention without interrupting the user's task.

The alert offers four severity levels that set a distinctive color.

Filled alerts

Filled alerts can be created by adding class alert with other classes alert--success, alert--error, alert--warning, alert--info.
