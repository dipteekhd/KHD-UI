Avatar

Avatars are user profile pictures,that can be of different shapes and sizes.

Circle Avatar

Circle avatars can be created by adding class rounded-circle with other classes avatar--sm, avatar--md, avatar--lg for small,medium,large avatars respectively.

Square Avatar

Square avatars can be created by adding class rounded-sm with other classes avatar--sm, avatar--md, avatar--lg for small,medium,large avatars respectively.

Shadow Avatar

Shadow on avatars can be created by adding class shadow with other classes avatar--sm, avatar--md, avatar--lg for small,medium,large avatars respectively.

Letter Avatar

Letter avatars can be created by adding classes avatar-letter, center-content with other classes avatar-letter--sm, avatar-letter--md, avatar-letter--lg for small,medium,large avatars respectively.
